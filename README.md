This is my desktop made with awesomewm! Here's some details:

- **Operating System** - endeavouros
- **Window Manager** - awesomewm
- **File Manager** - nautilus
- **Browser** - librewolf
- **Terminal** - st
- **Shell** - bash
- **Editor** - vim

## Installation

```
$ git clone https://github.com/stardust-kyun/dotfiles
$ dotfiles/install.sh
```
